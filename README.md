# 🌿 Paradise Nursery - Shopping Application

## 📌 Project Description
Paradise Nursery is an online plant shopping application where users can browse different types of houseplants, view their details, and add them to a shopping cart.

## 🚀 Features
- Browse plants by categories
- View plant details (image, name, price)
- Add plants to cart
- Increase/decrease quantity
- Remove items from cart
- Dynamic cart updates using Redux

## 🛠️ Technologies Used
- React.js
- Redux Toolkit
- JavaScript
- CSS
